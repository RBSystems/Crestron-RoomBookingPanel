# Crestron-RoomBookingPanel
DAL for Crestron TSS and TSW panels running Exchange Calendar Scheduling projects.

### Currently Monitored Statistics:
- ConnectionStatusMessage
- SerialNumber
- BuildDate
- CalendarSyncStatus
- PanelSyncing
- ExchangeUsername
- PufVersion
- ConnectionStatus
- MacAddres

### Current Controls:
- Reboot
